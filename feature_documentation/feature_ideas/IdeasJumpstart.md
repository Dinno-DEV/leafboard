# Main Functionality
- drag and drop mp3 file to add soundboard button.
- button has name label and unique soundboard volume.
- button has tags, can be filtered by tags, a button has max 5 tags, and a tag is maximum 16 letters.
- button can be searched by name label.
- button can be repositioned
- categories divide the buttons for all kinds of purpose.
- categories and buttons are saved everytime a button or category is added or modified.
- categories can be repositioned
- categories can be renamed
- resizeable window, scalable GUI.

# QoL Functionality
