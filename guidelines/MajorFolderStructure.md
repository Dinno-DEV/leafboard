# List of Major Folders:
assets, res_files, root_scenes, components, gui_components, singletons, 
libraries, custom_node_scripts, custom_resource_scripts, 
feature_documentation, guidelines.

# assets Folder
assets folder includes 2d, 3d, audio, video folders.

# res_files Folder
res_files are filled with .res files.

# root_scenes Folder
root_scenes are scenes that is meant to be the result of 
"get_tree().current_scene". They are typically levels or main menus.

# components Folder
components are filled with scenes that serves as component to another scene.
components scripts included.

# gui_components Folder
gui_components are filled with components but for gui. 
gui_components scripts included.

# singletons Folder
singletons are filled with global classes. 
singletons scripts included.

# libraries Folder
libraries are filled with classes with static functions. The scripts
in libraries are supposed to be used for utility purposes, think of it as
a toolbox, and the scripts are the tools.

# custom_node_scripts Folder
custom_node_scripts are filled with scripts that is purposed for custom node,
the nodes are not meant to be component scenes.

# custom_resource_scripts Folder
custom_resource_scripts are for scripts that inherits the Resource class.

# feature_documentation Folder
feature_documentation includes ideas to add, planned features, and documented
features of the application.

# guidelins Folder
guidelines include text files that helps on understanding the project structure
and conventions.
